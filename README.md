🔎 Project Overview

The React Recipe App interacts with the Meal DB API to provide a rich and dynamic recipe experience. Users can:

Search meals by name dynamically

View full recipe instructions and ingredients

Save favorite meals locally

Experience a fully mobile-responsive design

Navigate a clean, intuitive, and visually attractive interface

The app is designed to be fast, intuitive, and user-friendly, making it easy for users to explore new meals and maintain their favorite recipes.

🌟 Key Features
Feature	Description
Search Meals	Quickly search meals by name using a public API
Recipe Details	View ingredients, step-by-step instructions, and meal images
Favorites	Save favorite recipes locally in JSON for offline access
Responsive Design	Works seamlessly on mobile, tablet, and desktop
User-Friendly UI	Clean layout, recipe cards, and intuitive navigation

⚡ Technologies Used

React – Frontend library for building dynamic UI

CSS / Responsive Design – Mobile-first, adaptive layout

Meal API – Fetch meal data and recipes dynamically

JSON File Handling – Save and retrieve favorite recipes locally

🚀 Installation & Setup

Follow these steps to run the project locally:

Clone the repository

git clone https://github.com/yourusername/react-recipe-app.git
cd react-recipe-app


Install dependencies

npm install


Start the app

npm start


Open in browser
Navigate to http://localhost:3000
